Intercept attacks before they hit the disk. Guardy validates MIME types, checks cryptography, isolates threats via Scikit-Learn IsolationForest, and quarantines malicious uploads natively.

Why use Guardy?

We built this library so backend developers no longer have to rely solely on HTTP headers or file extensions.

  • Request Interceptor Validates constraints instantaneously to reject massive payloads.
  • Deep MIME Analysis Checks file bits instead of trusting the user's extension.
  • AI Anomaly Isolation Maps behavior using ML models like scikit-learn. 

# Install library
$ pip install guardy

# Import library
from guardy import FileAnalyzer

@app.post("/upload")
async def upload(file: UploadFile):
    assessment = analyzer.analyze_file(
        file_bytes=await file.read(),
        filename=file.filename
    )
    return assessment
